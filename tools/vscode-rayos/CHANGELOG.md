# Changelog

All notable changes to the RayOS VS Code extension will be documented in this file.

## [0.1.0] - 2026-01-09

### Added

- Initial release
- Build Kernel command with architecture selection
- Run in QEMU integration
- Create New App wizard with scaffolding
- Clean Build command
- Documentation browser
- Code snippets for App SDK:
  - `rayos-app` - Full app template
  - `rayos-descriptor` - AppDescriptor
  - `rayos-frame` - on_frame method
  - `rayos-event` - on_event method
  - `rayos-button` - Button drawing
  - Context drawing helpers
- Syntax highlighting for `.rayapp` manifest files
- Task provider for build/run/clean
- Problem matcher for Rust compiler errors
- Configuration options for QEMU and target arch
- Status bar integration
- Keyboard shortcuts (Ctrl+Shift+B for build, F5 for run)
